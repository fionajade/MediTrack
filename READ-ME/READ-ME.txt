MediTrack
Step 1: Download the MediTrack Folder
* Download the folder named MediTrack.
* Important Note: Do not alter the file directory structure or move the files under one folder. Any changes may result in a "URL not found" error when accessing the system.
Step 2: Start XAMPP Service
1. Launch XAMPP (if you haven't already installed it, you can download it from the official XAMPP website).
2. Start the Apache and MySQL services by clicking the "Start" button next to each service in the XAMPP Control Panel.
Step 3: Open phpMyAdmin
1. Open your browser and go to localhost or 127.0.0.1.
2. This will bring you to phpMyAdmin, where you can manage your databases.
o Ensure you are using phpMyAdmin version 5.2.1 for compatibility with the system.
Step 4: Create a New Database
1. In the phpMyAdmin dashboard, click on the Databases tab at the top.
2. Create a new database by entering the name meditrack in the "Database name" field.
3. Click on the Create button.
Step 5: Select the ‘meditrack’ Database
1. After creating the database, find meditrack in the list of databases.
2. Click on the meditrack database to open it.
Step 6: Import the SQL File
1. With the meditrack database selected, click the Import tab at the top of the screen.
2. Under the File to Import section, click on the Choose File button and select the meditrack.sql file from the folder you downloaded.
3. Click the Go button to begin importing the database. This will create all the necessary tables and structure for the MediTrack system.
Accessing the MediTrack System (MediTrack Login)
You can now log in to the MediTrack system and start using it. Note that there’s no separate URL for client and admin homepages; both use the same login interface.

Admin Login
* Username: admin
* Password: admin

Once logged in as admin, you will have access to the admin dashboard where you can manage the system, users, and other administrative functions.
Client Login
